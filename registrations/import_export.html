Essential if you are going to start Import-Export Business in India

What is Import Export Code (IEC)?

IEC is a 10-digit code issued by the Directorate General of Foreign Trade (DGFT) required for import-export transactions. You can obtain an IEC in your personal name or company name. An IEC number issued to an applicant is valid for all its divisions, units, factories, and branches. IEC is valid for a lifetime and does not require renewal.

Benefits of Import Export Code

  • Do Export Import Business: An IEC is necessary for the import/export of goods.
  • Avail Export Incentives: Companies can avail several benefits from DGFT, Export Promotion Council, SEIS Scheme, Customs, etc.

Required Documents

  • Passport Size Photograph (Soft Copy) of applicant (Proprietor/Director/Partner)
  • Identity proof of applicant (Proprietor/Director/Partner) (Passport, Driving License, Aadhaar card, Voter ID)
  • Cancelled Cheque for Support of Bank Details
  • PAN card of Company/Firm/LLP/Proprietor
  • AOA and MOA (In case of company)
  • Business Address Proof (Utility Bill/Lease Agreement/Sales Deed if owned property)
  • Certificate of Incorporation (In case of company)
  • Certificate of Registration (In case of Trust/Society)
  • Partnership Deed (In case of Firm)

Import Export Code Registration Process

Create an Online IEC Application

Visit the DGFT website at dgft.gov.in and click on “IEC Online Application”.

Add IEC Master Details

Select “Create” option from the “File” menu and fill in the details step by step.

Payment of Fee (Online)

Pay the fee online once the application is filled.

Upload Documents

Upload the required documents one by one after payment.

Submit to Regional Authority

Select the Regional Authority (DGFT) and submit the application.

Important Features of Import Export Code

  • Export Scheme Benefit: Required for all importers and exporters to obtain benefits.
  • No Return Filing: IEC code holders do not need to file any returns.
  • Lifetime Valid: The IEC code is issued for a lifetime and does not require renewal.
  • Individual Can Obtain: Even individuals who are proprietors can obtain the code in their name; it's not necessary to set up a business for this.
  • Single Code: The IEC issued is valid for all the branches or factories of the importer/exporter.

When is Import Export Code Mandatory?

  • When an importer has to clear shipments from customs.
  • When an importer sends money abroad through banks.
  • When an exporter has to send shipments.
  • When an exporter receives money in foreign currency directly into their bank account.

Frequently Asked Questions (FAQs)

Who must obtain Import Export Code? +
Any person who intends to import or export goods and services from or to India must obtain Import Export Code.
Where to file application to get Import Export Code? +
For Import Export Code, application must be filed online with the Regional Authority of DGFT of your business location.
How long does it take to obtain Import Export Code? +
After submission of all required documents and application, it takes 10-15 days.
Is there any monthly returns required to be filed with DGFT? +
Import Export Code is just a license allowing the import and export of goods. Therefore, no return is required to be filed with DGFT or with any other authority.
What is the validity of Import Export Code? +
Import Export Code is valid for a lifetime.
Do I need a separate Import Export Code for each branch office and factory? +
No. One Import Export Code is valid for all divisions.
Do I need to be physically present at the DGFT office to obtain Import Export Code? +
No. The process is online, and the IEC is issued and sent to your registered email ID and physically dispatched to your business address.
What is the procedure for depositing the fee for the online IEC application? +
Application fee must be deposited through online payment methods.
How can I modify my IEC application? +
An application can be modified any number of times until it is submitted for processing. You need to log in with the same details like PAN, mobile number, and email again.
